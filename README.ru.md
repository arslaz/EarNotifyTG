# EarNotifyTG

## Проблема

Вы в наушниках, телефон в кармане, экран выключен. Сообщение в Telegram — а от кого?
Приходится доставать телефон, разблокировать, проверять. Каждый раз.

## Решение

EarNotifyTG озвучивает имя отправителя прямо в Bluetooth-наушники.
Музыка приглушается, вы слышите "имя", музыка возвращается.
Телефон не трогали.

## Возможности

- 🔊 Озвучка имени отправителя Text-to-Speech
- 🎧 Работает только с Bluetooth-наушниками
- 📱 Только при выключенном экране
- 🔇 Автоматически приглушает и возвращает музыку
- ⚡ Системный сервис — работает всегда, не жрёт батарею

## Установка

1. Скачать APK из [Releases](../../releases)
2. Установить
3. Включить доступ к уведомлениям (приложение подскажет)
4. Нажать кнопку **ENABLED**
5. Подключить Bluetooth-наушники

## Требования

- Android 8.0+
- Bluetooth-наушники
- Telegram установлен

## Технологии

- Kotlin
- Jetpack Compose
- NotificationListenerService
- TextToSpeech
- AudioManager
